## Shortcuts

Rename HTML tag:

    Ctrl + Shift + U
    
Rename JS variable/function:

    Ctrl + Alt + r

Go to matching HTML tag:

    Ctrl + Alt + j

Find a file:

    Cmd + T

Go to line:

    Ctrl + G

Cut move line:

    Ctrl + Cmd + [up]
    Ctrl + Cmd + [down]
  
Comment current selection/line:

    Cmd + /
  
Select current line:

    Cmd + l
    
Color picker:
    
    Cmd + Shift + C
